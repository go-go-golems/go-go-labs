Focus Stack

User Guide

Welcome to Focus Stack, a tool for capturing multiple images at different focus depths.

How to use:

  1. Select Camera: Choose your preferred camera from the dropdown menu.
  2. Choose Resolution: Select the desired resolution for your captures.
  3. Set Frame Count: Decide how many frames you want to capture.
  4. Adjust Countdown Speed: Set the time between each capture.
  5. Start Capture: Click the "Start Capture" button to begin the process.
  6. Wait for Completion: The app will countdown and capture the specified number of frames.
  7. Download Images: Each captured frame will be automatically downloaded to your device.

Note: Ensure you grant camera permissions when prompted by your browser.

Progress Bars and Countdown

During the capture process, you'll see two progress bars:

  • Countdown Bar (Orange): This shows the countdown between each frame capture. It will count down from the specified countdown speed to 0 before each capture.
  • Progress Bar (Green): This indicates the overall progress of the capture session. It fills up as each frame is captured, showing you how many frames have been captured out of the total.

The countdown bar will reset for each frame, while the progress bar will continue to fill until all frames are captured.

File Naming Convention

The captured images are automatically downloaded with the following naming convention:

frame_[run_number]_[frame_number].png

  • run_number: Increments each time you start a new capture session.
  • frame_number: A three-digit number (padded with zeros if necessary) indicating the frame's position in the current capture sequence.

For example, the third frame of your second capture session would be named: frame_2_003.png
