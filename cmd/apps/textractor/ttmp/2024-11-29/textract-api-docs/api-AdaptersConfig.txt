AdaptersConfig

   Contains information about adapters used when analyzing a document,
   with each adapter specified using an AdapterId and version

Contents

   Adapters
          A list of adapters to be used when analyzing the specified
          document.

          Type: Array of Adapter objects

          Array Members: Minimum number of 1 item. Maximum number of 100
          items.

          Required: Yes
