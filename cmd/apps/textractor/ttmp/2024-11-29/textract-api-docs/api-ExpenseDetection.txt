ExpenseDetection

   An object used to store information about the Value or Label detected
   by Amazon Textract.

Contents

   Confidence
          The confidence in detection, as a percentage

          Type: Float

          Valid Range: Minimum value of 0. Maximum value of 100.

          Required: No

   Geometry
          Information about where the following items are located on a
          document page: detected page, text, key-value pairs, tables,
          table cells, and selection elements.

          Type: Geometry object

          Required: No

   Text
          The word or line of text recognized by Amazon Textract

          Type: String

          Required: No
